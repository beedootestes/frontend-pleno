## Instruções:

- Clone este repositório
- Entre na pasta do projeto

```bash
cd frontend/project
```

- Instale as dependências

```bash
npm install
```

- Rode o projeto

```bash
npm run dev
```

- Acesse o projeto em http://localhost:5173

- Execute os testes

```bash
npm run test:e2e
```
